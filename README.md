# FLOMI Studio

> Follow me, follow the ocean.

FLOMI 浮米海洋戶外工作室 — 台灣海岸生活風格品牌平台。

## 關於本站

本站採用單頁靜態網站架構,以編輯雜誌 × 紀錄片電影感的美學語言,呈現 FLOMI 的品牌定位、核心價值、服務內容、Journal、合作夥伴與發展藍圖。

## 技術

- 純 HTML / CSS,無框架依賴
- 字體:Fraunces、Noto Serif TC、JetBrains Mono(Google Fonts)
- 響應式設計(桌機 / 行動裝置)

## 本機預覽

直接用瀏覽器開啟 `index.html` 即可,或:

```bash
python3 -m http.server 8000
# 瀏覽器開啟 http://localhost:8000
```

## 部署到 GitHub Pages

在 repo 設定 → Pages → Source 選 `main` branch / `/ (root)`,等一兩分鐘即可上線。
